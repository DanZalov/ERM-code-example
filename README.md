# ERP-code-exmple

## Short Description

Part of a frontend side for ERP-system app (SPA)

## Content

–ê front page is linked to a few additional ones. There are some forms with validation through bootstrap and react-hook-form. App has no backend but ready for it (RTKQuery templates). Login, time register, add employee, customer and project editor pages can be used in connection with backend.

## Development info

- Typescript
- React
- Redux toolkit + RTK Query
- Bootstrap
- react-hook-form and bootstrap validation
